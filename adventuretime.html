→ Now let's enter the browser and run “https://Machine IP”

“I've lost Jake, can you help me find him?” I decided to scan the directory and started the scan using the dirb tool

https://MachineIP/candybar/

When we found this directory and looked at the site, we were greeted with a screen like this
